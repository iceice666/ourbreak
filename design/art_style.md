# Art Style Document — ourcraft

基於 `gdd.md` 的世界觀與角色設定，定義 ourcraft 的視覺語言。

---

## 1. 整體美術方向

**風格定向：熱帶卡通低多邊形（Tropical Cartoon Low-Poly）**

- 低多邊形（Low-poly）幾何感，稜角乾淨，模型面數控制在可管理範圍內
- 鮮豔、飽和的熱帶配色，高對比度，讓遊戲讀圖清晰
- 角色造型圓潤、表情誇張，帶有「可愛但衝突」的喜劇感
- 整體視覺氛圍：明亮、輕鬆、有趣——沙灘上的小鬧劇，而非嚴肅戰鬥

**設計原則**

| 原則 | 說明 |
|------|------|
| 可讀性優先 | 不同方塊、武器的視覺輪廓必須在第一人稱視角下一眼辨別 |
| 色彩編碼 | 每種方塊有專屬主色，玩家不需看文字就能辨認 |
| 世界一致性 | 所有物件（角色、方塊、武器、地形）共享同一套光影與輪廓語言 |
| 效果清晰 | 特殊效果（減速、反彈、閃爍）的視覺回饋必須即時且明確 |

## 2. 色彩系統

### 2.1 主色板

| 色彩名稱 | Hex | 用途 |
|----------|-----|------|
| 沙灘金 | `#F4D56A` | 地板主色、Sand Block |
| 熱帶藍 | `#3AB8E0` | 天空、海水、UI 強調色 |
| 珊瑚粉 | `#FF6B6B` | Clawd 主色、Shell Block |
| 開源紅 | `#D93E2E` | Openclaw 主色、玩家 HUD 強調 |
| 礁石灰 | `#8E9A9A` | Rock Block |
| 珊瑚橙 | `#FF9F45` | Coral Block |
| 水母藍紫 | `#9B7FD4` | Jellyfish Block |
| 椰葉綠 | `#4CAF50` | 背景植被 |
| 純白沙 | `#FFF8E7` | UI 背景、高光 |

### 2.2 色彩使用原則

- 每個方塊類型以**一種主色 + 一種輔色**構成，不超過兩色
- 玩家 UI 元素使用開源紅系；敵方（Clawd）相關元素使用珊瑚粉系
- 天空、遠景維持**低飽和度**，避免搶奪近景焦點

## 3. 角色美術

### 3.1 Clawd（粉紅色章魚 — 建造者 / 閉源派）

| 項目 | 規格 |
|------|------|
| 主色 | 珊瑚粉 `#FF6B6B` |
| 輔色 | 淡粉白（腹部）、深粉紅（觸手末端）|
| 體型 | 矮胖圓潤，觸手短而蓬鬆，眼睛大而圓 |
| 個性表現 | 表情帶有傲嬌感，靜止時微微交叉觸手 |
| 比例 | 頭身比約 1:1（Q版），全高約 1.5 格 |
| 細節 | 腹部有隱約閉鎖圖騰裝飾（對應「閉源派」）|

### 3.2 Openclaw（紅色龍蝦 — 破壞者 / 開源派）

| 項目 | 規格 |
|------|------|
| 主色 | 開源紅 `#D93E2E` |
| 輔色 | 橘紅（甲殼高光）、米白（腹部）|
| 體型 | 略高於 Clawd，螯大而有力，姿態積極前傾 |
| 個性表現 | 表情自信爽朗，持武器時會有誇張舉起的動作 |
| 比例 | 頭身比約 1:1.2，全高約 1.5 格 |
| 細節 | 螯上有小型開鎖 / 開源齒輪裝飾（對應「開源派」）|

### 3.3 動畫方向（參考）

| 動畫 | Clawd | Openclaw |
|------|-------|----------|
| 閒置 | 觸手輕晃，微微上下浮動 | 螯輕微夾動 |
| 建造 / 攻擊 | 觸手飛速放方塊 | 根據武器播放揮砍 / 射擊動畫 |
| 受勝利事件 | 跳腳生氣 | 舉螯歡呼 |

## 4. 方塊美術

所有方塊共享統一基底：**約 1×1×1 格的低多邊形立方體**，上面有代表該方塊特性的紋理與輪廓裝飾。

| 方塊 | 主色 | 視覺特徵 | 破壞特效 |
|------|------|----------|----------|
| Sand Block | 沙灘金 `#F4D56A` | 粗糙沙粒紋理，稍微不規整的邊角 | 沙塵飛散（黃色粒子）|
| Coral Block | 珊瑚橙 `#FF9F45` | 表面有分叉珊瑚形狀突起，邊緣帶橘紅暈 | 珊瑚碎屑（橘粒子）+ 淡藍減速光環 |
| Shell Block | 珊瑚粉 `#FF6B6B` | 螺旋貝殼紋路，帶光澤感（模擬反光） | 爆裂閃光（白色衝擊波）+ 彈片粒子 |
| Rock Block | 礁石灰 `#8E9A9A` | 不規則多邊形切面，表面有裂痕紋路 | 石塊碎裂（灰色粒子，較大碎片）|
| Jellyfish Block | 水母藍紫 `#9B7FD4` | 半透明外殼，內部有脈動光點（idle 動畫） | 爆散水母膠（藍紫粒子）|

**方塊破損視覺：**  
耐久消耗時，方塊表面出現裂痕貼圖（依剩餘耐久分 3 個破損等級），Rock Block 裂痕最明顯。

## 5. 武器美術

武器在第一人稱視角中顯示於畫面右下角（持握視角），風格與角色一致——卡通誇張，輪廓清晰。

| 武器 | 視覺描述 | 攻擊特效 |
|------|----------|----------|
| Sword（劍） | 扁平寬刃海盜式短劍，刀刃有沙灘波浪紋，握柄有螃蟹鉗造型 | 揮砍產生橘紅色弧形軌跡殘影 |
| Gun（槍） | 圓潤塑料感水槍（海灘風），槍身鮮豔黃色，有波浪貼紙裝飾 | 發射時水柱粒子 + 白色撞擊閃光 |
| Drone（無人機） | 迷你沙灘巡邏風格，四旋翼，機身黑白配色，有小型觀光相機 | 俯衝時螺旋氣流粒子，爆炸時橘黃煙霧球 |

## 6. 場景美術

### 6.1 地板

| 材質 | 描述 |
|------|------|
| 沙地 | 主要地面，淺黃色，帶細緻波紋壓痕，低多邊形呈現沙堆起伏 |
| 礁石 | 點綴於沙地間，礁石灰，不規則形狀，玩家不可站立 |

### 6.2 背景（固定場景，非互動）

| 元素 | 描述 |
|------|------|
| 天空 | 漸層天藍（`#87CEEB` → `#3AB8E0`），無雲或少量卡通積雲 |
| 海浪 | 遠景，藍白色低多邊形波浪，有緩慢起伏動畫 |
| 椰子樹 | 2–3 棵點綴，葉片卡通化，有微風搖曳動畫 |
| 太陽 | 畫面右上角，圓形卡通太陽，帶輻射光芒 |

### 6.3 場景色調

- 整體亮度高，陰影柔和（半影），強調輕鬆感
- 禁止使用深色或陰暗場景元素（與世界觀不符）

## 7. 特殊效果（VFX）

| 效果 | 觸發條件 | 視覺描述 |
|------|----------|----------|
| Coral 減速光環 | 玩家進入 Coral Block 1.5 格範圍 | 玩家腳邊藍綠色光環脈動，每 0.5 秒閃一次 |
| Shell 反彈爆炸 | Shell Block 被摧毀 | 白色衝擊波從方塊中心向外擴散，2–3 枚粉色彈片飛出 |
| Jellyfish 視野干擾 | Jellyfish Block 被放置後 | 全螢幕藍紫色閃爍 overlay，邊緣模糊，持續 TBD 秒 |
| Drone 爆炸 | Drone 引爆 | 3×3 橘黃煙霧球，持續約 0.5 秒後消散 |
| 方塊破壞 | 任何方塊耐久歸零 | 依方塊主色噴出粒子，伴隨小型音效（對應 VFX 章節）|

## 8. UI / HUD 美術

### 8.1 設計語言

- **扁平卡通（Flat Cartoon）**：圓角矩形邊框，無複雜漸層，色塊簡潔
- 字體：粗體無襯線字（推薦：Nunito Bold 或類似圓潤風格），英文標籤 + 數字顯示
- 邊框裝飾元素取材自沙灘（沙堡、貝殼、小波浪）

### 8.2 HUD 佈局

```
┌─────────────────────────────────────────┐
│ [Round 1/4]              [Timer: 0:60]  │
│                       [Buildings: 12]   │
│                                         │
│                                         │
│                    ✛                    │  ← 準心
│                                         │
│                                         │
│                [1] [2] [3]              │  ← 武器槽
└─────────────────────────────────────────┘
```

| 元素 | 顏色 | 描述 |
|------|------|------|
| Round 計數 | 白字 + 沙棕背景框 | 左上角，常駐顯示 |
| 計時器 | 白字 + 熱帶藍背景框 | 右上角，ATTACK phase 顯示，< 10 秒轉紅並閃爍 |
| 建築數量 | 白字 + 珊瑚粉背景框 | 右上角計時器下方，ATTACK phase 顯示 |
| 武器槽 | 選中高亮金框，未選灰框 | 下方中央，三格，顯示武器圖示 |
| 準心 | 白色十字 | 螢幕正中央 |

### 8.3 主選單 / 結算畫面

| 畫面 | 風格 |
|------|------|
| 主選單 | 全螢幕沙灘背景（概念圖 `ourcraftconcept.png` 風格），大型 OURCRAFT 標題字，Start Game / Exit 圓角按鈕 |
| 選角畫面 | Clawd 與 Openclaw 的立繪並排，選中角色有光圈高亮 |
| 勝利畫面 | 橘金色「YOU WIN!」大字 + Openclaw 歡呼動畫，Restart 按鈕 |
| 失敗畫面 | 藍灰色「YOU LOSE...」大字 + Openclaw 垂頭動畫，Restart 按鈕 |

## 9. 燈光與氛圍

| 光源 | 類型 | 描述 |
|------|------|------|
| 主光源 | 方向光（Directional Light）| 模擬晴天正午陽光，角度 45°，淡黃色 `#FFF5C0` |
| 環境光 | Ambient Light | 低強度天藍色 `#C9E8F5`，填補陰影死角 |
| 方塊特效光 | 點光源（可選）| Jellyfish Block 放置時，附近散發藍紫色微光 |

- 無動態全局陰影（效能考量），僅角色與物件有靜態貼花陰影（blob shadow）
- 整體場景色溫偏暖（陽光），強調「下午沙灘」的氛圍
